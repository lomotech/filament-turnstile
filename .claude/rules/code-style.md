# Code Style

- PHP 8.2+ with `declare(strict_types=1)`
- Use PHP 8 attributes instead of docblock annotations
- PSR-4 autoloading under `l3aro\FilamentTurnstile` namespace
- Format with Laravel Pint ("per" preset)
- PascalCase for classes, camelCase for methods/properties, SCREAMING_SNAKE_CASE for constants
- Group imports by type (classes, functions, constants) with blank lines between groups
- 4-space indentation, UTF-8, LF line endings
